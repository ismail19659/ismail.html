HTML & CSS Quick Reference Guide

HTML & CSS Quick Reference Guide

This guide covers various HTML tags and CSS properties, with code examples.

HTML Tags

<h1> to <h6>

The <h1> to <h6> tags are used for headers. <h1> is the highest level header, and <h6> is the lowest.

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>

<p>

The <p> tag is used to define a paragraph.

<p>This is a paragraph of text.</p>

<a>

The <a> tag is used to create links.

<a href="https://example.com">Visit Example</a>

<ul> and <li>

The <ul> tag defines an unordered list, and <li> defines list items.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol> and <li>

The <ol> tag defines an ordered list, and <li> defines list items.

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

<img>

The <img> tag is used to embed images.

<img src="image.jpg" alt="An image">

<div>

The <div> tag is a block-level container used for grouping content.

<div>
    <p>Some content inside a div</p>
</div>

<span>

The <span> tag is an inline container used for grouping inline content.

<span>This is inline text.</span>

<form>

The <form> tag is used to create an HTML form.

<form action="submit_form.php" method="POST">
    <input type="text" name="name">
    <input type="submit" value="Submit">
</form>

<table>, <tr>, <td>

The <table> tag is used to create a table, <tr> for table rows, and <td> for table data.

<table>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

CSS Properties

font-family

The font-family property is used to define the font of an element.

p {
    font-family: Arial, sans-serif;
}

color

The color property is used to change the text color.

p {
    color: blue;
}

background-color

The background-color property is used to change the background color of an element.

div {
    background-color: #f0f0f0;
}

margin

The margin property is used to create space around an element.

div {
    margin: 20px;
}

padding

The padding property adds space inside an element, between its content and its border.

div {
    padding: 15px;
}

border

The border property is used to add a border around an element.

div {
    border: 1px solid black;
}

text-align

The text-align property is used to align text horizontally.

p {
    text-align: center;
}

display

The display property specifies the display behavior of an element.

div {
    display: flex;
}
